# 绿联 NAS Docker 部署说明（纯 UI 操作 · 镜像拉取）

适配机型：**DXP4800 Plus** (Intel 8505 x86_64)
其他绿联 x86 NAS 同样适用。

---

## 部署步骤（3 步）

### 1. 准备文件

在 NAS 文件管理器中创建项目目录，如 `/volume1/docker/wow/`，放入以下内容：

```
docker/wow/
├── data/                          # 客户端地图数据（见下方说明）
│   ├── maps/
│   ├── dbc/
│   ├── vmaps/
│   ├── mmaps/
│   └── cameras/
├── scripts/                       # 从 nas/scripts/ 复制
│   ├── realm-init.sh
│   └── create-default-account.py
└── .env                           # 可选（不需要也可以部署）
```

> 客户端地图数据约 2-3GB，用 AzerothCore 提取工具从 3.3.5a 客户端导出。

### 2. 编辑 Compose 配置

打开 `nas/docker-compose.yml`，**只需要改一个地方**：

搜索 `REALM_IP`，把 `192.168.1.100` 改成 NAS 的局域网 IP：

```yaml
ac-realm-init:
    environment:
      REALM_IP: 192.168.31.11   # ← 改成你的 NAS IP
```

如果需要改数据库密码，搜索 `wow@asm0x1` 全部替换（**所有出现的地方都要改，共 5 处**）。

### 3. 部署

1. 绿联 NAS 桌面 → **Docker** → **项目** → **新建项目**
2. 项目名称：`wow`
3. 项目文件夹路径：选择上面创建的目录（如 `/volume1/docker/wow/`）
4. 将修改后的 `nas/docker-compose.yml` **全部内容粘贴**到编辑器
5. 点击 **「部署」**

首次部署自动拉取镜像，约 5-10 分钟（取决于网速）。

## 确认运行状态

部署完成后，在 Docker → 容器页面查看：

| 容器 | 预期状态 |
|------|----------|
| acore-database | ✓ 运行中 |
| acore-worldserver | ✓ 运行中 |
| acore-authserver | ✓ 运行中 |
| acore-registration | ✓ 运行中 |
| acore-realm-init | 已退出(0) ← 正常 |
| acore-account-init | 已退出(0) ← 正常 |

### 查看 Realm IP 是否正确

点击 `acore-realm-init` → 日志，确认看到：

```
>> [realm-init] 最终配置: 192.168.31.11:8085
>> [realm-init] Realm 地址已设置完成
```

如果显示 `127.0.0.1`，说明 YAML 中的 `REALM_IP` 没改，回到步骤 2。

## 客户端登录

1. 魔兽世界 3.3.5a 客户端 → `Data/zhCN/realmlist.wtf` 改为：
   ```
   set realmlist NAS的局域网IP
   ```
2. 运行 `Wow.exe` 登录
3. 内置账号：`asm0x1` / `123456`
4. Web 注册：浏览器访问 `http://NAS_IP:8765`

## GM 命令（在 Docker → acore-worldserver → 终端 中执行）

```
account create 用户名 密码          # 创建账号
account set gmlevel 用户名 3 -1     # 设为管理员 (GM 3)
account onlinelist                  # 查看在线玩家（含假人）
```

---

## 调整假人数量

编辑 YAML 中 `ac-worldserver` 的 `environment` 部分，修改后重新部署：

```yaml
BOT_MIN_COUNT: 10
BOT_MAX_COUNT: 30
```

| NAS 内存 | 建议假人数 |
|----------|------------|
| 4GB | 5-10 |
| 8GB | 15-30 |
| 16GB+ | 30-100 |

---

## 更新镜像

1. Docker → 容器 → 删除 `acore-worldserver`、`acore-authserver`、`acore-registration`
2. Docker → 镜像 → 删除旧版 `asm0x1/wow-*` 镜像
3. Docker → 项目 → wow → 重新部署

---

## 完全重置

清除所有数据：

1. Docker → 项目 → wow → 操作 → **删除项目**
2. **勾选「同时删除卷」**
3. 重新创建项目部署

---

## 防火墙

如 NAS 启用了防火墙，放行：

| 端口 | 说明 |
|------|------|
| 3724 (TCP) | 游戏登录 |
| 8085 (TCP) | 游戏世界 |
| 8765 (TCP) | Web 注册 |

---

## 可选：启用 phpMyAdmin

编辑 YAML，取消 `phpmyadmin` 段的注释，重新部署。访问 `http://NAS_IP:8081`。
